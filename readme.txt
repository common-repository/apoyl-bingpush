== Description ==
这是一款解决把你文章内容推送到Bing必应搜索引擎里，也可以推送到IndexNow，让Bing第一时间抓取你的内容，加速Bing收录（必应收录）你的网站内容。IndexNow是让搜索引擎及时索引新技术，帮助更好实现网站SEO，让搜索引擎更快的收录。

## 功能概述

* 支持点击文章 推送到Bing搜索引擎收录里(Bing推送URL提交SEO-Bing Push)
* 支持后台自定义域名和密钥
* 支持对应https
* 支持自动推送bing,无需要人工干预+
* 支持新发布文章自动推送到bing和老文章自动推送到bing
* 支持采集回来文章自动推送Bing搜索引擎收录里
* 新增支持自动推送IndexNow SEO优化，无需要人工干预[IndexNow官网](https://www.bing.com/indexnow)
